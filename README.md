## 介紹
一個可視化調整UI的工具，將現成的UI放入後可以快速進行調整，並支援Risu平台的regex直接匯入匯出。
能透過快捷鍵在預覽區快速定位選取元件的css或html位置，或是變數區的放大鏡定位，省去肉眼搜尋的時間。
可以將css常見的大小、間距、顏色等等設定成拉條控制。

## 🎨 **變數控制台** (左側面板上方)
### **智慧提煉**按鈕 🧙‍♂️
自動掃描你的樣式表,把所有數值轉換成可調整的滑桿:
- **顏色** → 色盤選擇器
- **尺寸** → 拖曳滑桿(如字體大小、間距)
- **角度** → 旋轉控制
- **視窗單位** → 響應式尺寸

**使用情境:**
> 你寫了一段樣式,但想快速測試不同的顏色搭配或間距效果,不用一直修改數字。點擊「智慧提煉」後,所有數值變成滑桿,拖動就能即時看到變化!

### **燒錄儲存**按鈕 🔥
當你調整滿意後,把滑桿的數值永久寫回樣式表,並清空控制台。

### **篩選器**
可選擇要提煉哪些類型的數值:
- ✅ 顏色
- ✅ 尺寸
- ✅ 角度
- ✅ 視窗單位



## 🧪 **RISU 變數模擬** (左側面板第二區)
這是專為 RisuAI 平台設計的功能,可以模擬角色卡中的動態變數。

### **掃描**按鈕 📡
自動偵測你的內容中使用了哪些變數,例如:
- `{{getvar::player_name}}` → 角色名稱
- `{{getvar::player_level}}` → 等級
- `{{image::character_avatar}}` → 角色頭像

掃描後會列出所有變數,你可以填入測試用的數值。

### **手動新增**按鈕 ➕
自行新增變數進行測試,例如:
- 變數名稱:`player_hp`
- 預設值:`75`

### **上傳圖片**按鈕 📤
上傳自訂圖片用於測試圖片資產的顯示效果,系統會自動偵測圖片尺寸。



## 📝 **CSS 片段庫** (左側面板第三區)

### **CSS 片段**分頁
預設提供 40+ 個常用樣式效果,點擊即可插入,支援手動新增樣式並匯出:

**基礎樣式:**
- 🔳 圓角、📏 邊框、📦 內距/外距

**文字排版:**
- 🔤 字體大小、🅱️ 字體粗細、✂️ 文字截斷

**顏色與背景:**
- 🎨 文字顏色、🖌️ 背景顏色、🌈 背景漸層、✨ 文字漸層

**佈局容器:**
- 📦 Flex 容器、📲 Grid 容器、📍 絕對定位

**陰影與效果:**
- 👻 盒子陰影、💎 文字陰影、🪟 玻璃擬態

**互動效果:**
- 👆 滑鼠手勢、🎭 過渡動畫、🔍 懸浮放大、💫 關鍵幀動畫

### **響應式輔助**分頁
先用「選取模式」點擊預覽區的元素,再點擊效果按鈕:

**手機版效果 (≤768px):**
- 🙈 隱藏、🔤 縮小字體、↔️ 滿版寬度
- ⬇️ 直向排列、📋 置中文字、📏 移除內距

**平板版效果 (769px~1024px):**
- 🙈 隱藏、🔲 兩欄佈局、🔤 稍微縮小


## ✏️ **編輯器區** (左側面板下方)

### **四個編輯器分頁、兩個編輯組別:**
css+html為一組，regex+劇情為一組，兩邊的變數互不干擾。

### **📄 CSS 分頁**
- 支援多個 CSS 分組(雙擊分頁開啟選單管理)
- 可設定「全域 CSS」(套用到所有HTML頁面)
- 可設定「獨立 CSS」(僅對連結的 HTML 生效)
- 右下角有「提煉選取」按鈕(從選取的程式碼片段單獨提煉變數)

### **📄 HTML 分頁**
- 支援多個頁面(雙擊分頁開啟選單管理)
- 可連結特定 CSS 分組


### **📄 Regex 分頁** 
左側規則列表 + 右側編輯器結構:
- **規則列表:** 顯示所有轉換規則,可選擇是否對REGEX模式預覽區顯示內容。
- **編輯器:** 設定規則名稱、類型、標籤、輸入模式(IN)、輸出模板(OUT)

### **📄 劇情分頁**
輸入模擬劇情文本,用於測試 Regex 規則的轉換效果。



## **即時預覽區** (右側面板)
- **兩種預覽模式:**

### **🔹 簡易模式** (預設)
直接渲染 CSS + HTML 的內容,適合一般開發。

### **🔹 Regex 模式**
套用 Regex 規則處理劇情後渲染,適合Risu平台開發者。
- 支援:   
`gi<move_top>` = 移動到頂部
`gi<move_bottom>` = 移動到底部


## 💾 **工作狀態管理** (設定選單)

### **匯出進度**按鈕
將整個工作環境打包成 JSON 檔案,包含:
- ✅ CSS 分組
- ✅ HTML 頁面
- ✅ Regex 規則
- ✅ 變數狀態
- ✅ 劇情內容
- ✅ 編輯器和預覽狀態
- 🔲 上傳的圖片(可選)

### **輸出處理**
- 合併複製與匯出regex時，自動清理多餘的換行，避免造成UI變形。

### **匯入進度**按鈕
載入之前儲存的工作環境,一鍵恢復所有設定。

**使用情境:**
> 你正在製作角色卡,做到一半想換電腦繼續。點擊「匯出進度」下載 JSON 檔案,換到另一台電腦後「匯入進度」,所有內容立刻恢復!




## 📤 **合併複製功能** (頂部按鈕)

### **智慧合併邏輯:**
1. 選擇要複製的 HTML 頁面
2. 勾選要包含的 CSS:
   - ✅ 包含全域 CSS
   - ✅ 包含連結 CSS
3. 自動組合成完整的 HTML 檔案

系統會智慧處理:
- 如果 CSS 中已有 `<style>` 標籤 → 直接使用
- 如果沒有 → 自動包裹 `<style>` 標籤
- 自動清理多餘空行(可在設定中關閉)

**使用情境:**
> 你完成一個UI頁面,想複製到其他平台。點擊「合併複製」,選擇該要複製的HTML,勾選相關 CSS,一鍵複製完整程式碼,直接貼到平台即可!



## ⏪ **Undo/Redo 功能**

### **桌機版:**
- **Ctrl + Z** → 復原變數調整 / CSS 編輯
- **Ctrl + Y**、**Ctrl + Shift + Z** → 重做

### **說明:**
系統會記錄:
- 變數控制台的調整歷史(最多 50 筆)
- CSS 編輯器的拖曳控制操作(最多 30 筆)


## **👍實用功能**
### 🔍 **搜尋過濾**
- 變數控制台、RISU 變數皆有搜尋框
- 變數區皆可透過放大鏡按鈕快速定位需要的項目

### **拖曳式放大位移CSS元件**
在預覽區用Ctrl + 滑鼠左鍵會出現八個方向的拖曳鈕，分別對應:
- 寬高(四個角落)
- 邊距(四個邊緣)
### 請注意，如果你選擇的是偽元素，拖曳後是不會真的改變CSS。
